# LEDs
(Light Emitting Diodes) Electronic components that produce light when current flows through them.
## Single Colour LED
An LED that emits only one colour (e.g., red, green, or blue).
picture
## RGB LED (Common Anode Configuration)
An LED containing red, green, and blue LEDs in one package.

In a common anode setup, all positive terminals are connected, and colours are controlled by grounding the individual colour pins.
