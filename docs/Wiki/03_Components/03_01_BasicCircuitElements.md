# Circuit
A closed path that allows electric current to flow through components.
picture

## Integrated Circuit (IC)
A small electronic chip containing many components like transistors and resistors.
picture

## Circuit Elements

### Resistor
Adjusts the voltage and current flow in a circuit.
picture

### Potentiometer
A variable resistor that allows manual adjustment of voltage (often used as a dial).
picture

### Wire
A conductor used to connect components in a circuit.
Commonly, red wires are used for positive voltage and black wires for ground
picture

### Jumper Cables
Pre-made wires used to quickly connect components on a breadboard.
picture
