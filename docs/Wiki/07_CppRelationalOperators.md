# Comparisons

Used to compare variables (useful for if() statements)

| Operator     | Meaning                   |
|:-------------|:--------------------------|
| ==           | equal to                  |
| !=           | not equal to              |
| <            | less than                 |
| <=           | less than or equal to     |  
| >            | greater than              |
| >=           | greater than or equal to  |

{: .warning}
> Do not confuse == (equality) and = (assignment)
